# Jogo do NIM

## Descrição
Jogo do NIM implementado em Python 3, onde o computador utiliza uma estratégia vencedora baseada em múltiplos de (m+1).

## Como executar

```bash
python3 main.py
```

## Regras do Jogo

- N peças são dispostas no tabuleiro
- Dois jogadores alternam jogadas
- Cada jogador pode tirar de 1 até M peças por vez
- Quem tirar a última peça ganha
